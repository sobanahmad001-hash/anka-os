# Anka OS — Final Implementation Roadmap

## Goal
Turn the completed planning framework into a safe implementation order.

## Core Direction
Anka OS should be implemented as:
- Admin spine
- shared project core
- specialized environments for Development, Design, and Marketing
- environment-aware AI

## Implementation Sequence
1. Shell refactor
2. Admin environment refactor
3. Development environment refactor
4. Design environment refactor
5. Marketing environment refactor
6. AI behavior wiring
7. cleanup and deployment

## Important Rules
- do not delete legacy files too early
- do not mix schema-heavy changes with shell refactor
- do not rebuild everything in one code batch
- stabilize navigation before environment rewrites
- test after each batch

# Admin — Keep / Convert / Remove

## Keep
- user management
- role/access logic
- project visibility logic
- dashboard foundation
- task visibility logic

## Convert
- generic dashboard -> org command dashboard
- app-first navigation -> admin-spine navigation
- generic kanban -> execution support view
- layout/sidebar -> environment-aware admin structure

## Remove / Downgrade
- third-party app feel
- window-based workspace identity
- utility apps as equal first-class admin objects
- clutter that weakens portfolio visibility

# Anka OS — Phase 3 Development Environment

## Development Identity
Development is the technical execution environment of Anka OS.

## Development Purpose
Development helps the team manage implementation, sprint flow, technical blockers, code context, releases, and delivery progress.

## Development Owns
- active development projects
- sprint flow
- task execution
- technical blockers
- technical decisions
- linked specs/docs
- repo and deployment context

## Development Core Views
1. Dev Overview
2. Project Execution
3. Sprint / Work Queue
4. Technical Blockers
5. Technical Decisions
6. Code / Release Context

## Development Should Not Be
- just a terminal wrapper
- just a kanban board
- just a GitHub viewer
- just a generic dashboard with code widgets

## Development AI Role
Development AI is a technical execution helper.

It helps with:
- summarize project state
- explain recent changes
- identify blockers
- suggest next steps
- break down work
- summarize PR / deployment context
- connect tasks, blockers, and decisions
- surface technical risk

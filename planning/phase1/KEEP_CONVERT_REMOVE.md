# Keep / Convert / Remove

## Keep
- profiles
- tasks
- projects
- project progress logic
- department roles and hierarchy
- assignment logic
- admin oversight concepts

## Convert
- app shelf model -> environment model
- department apps -> department environments
- generic workspace -> project operating system
- shared apps as core -> project execution as core

## Remove / Downgrade
- window-based UX as main paradigm
- third-party-app style clutter
- making notes/wiki/templates primary
- expanding modules before shared core is stable
